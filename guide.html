Guide des outils numériques

Tout ce qu'il faut savoir pour encoder vos observations efficacement

Démarrage rapide — Outil bureau

1

Ouvrir l'outil

Accédez à l'outil bureau. La carte de la Gaume s'affiche avec les délimitations des entités en vert pointillé.

2

Cliquer sur un bâtiment

Cliquez précisément sur le bâtiment où se trouvent les nids. Le formulaire s'ouvre et le village ainsi que l'adresse sont détectés automatiquement en quelques secondes.

3

Remplir le formulaire

Dans l'ordre : vérifiez le village et l'adresse, sélectionnez le type de bâtiment, encodez le nombre de nids (5 colonnes), cochez l'emplacement, ajoutez une remarque si nécessaire.

Le curseur se place automatiquement sur « Nids entiers » à l'ouverture. Vous pouvez encoder directement au clavier et naviguer entre les colonnes avec la touche Entrée.
4

Enregistrer l'observation

Cliquez sur Ajouter l'observation. Un point apparaît sur la carte. Les données sont sauvegardées localement et envoyées automatiquement vers la base de données.

5

Exporter le CSV

Une fois votre session terminée, cliquez sur Télécharger dans la barre du haut. Le fichier CSV est nommé automatiquement selon le village et l'année. Transmettez-le à la personne de contact.

Démarrage rapide — Outil terrain

1

Ouvrir sur votre téléphone

Ouvrez l'outil terrain dans votre navigateur mobile. La carte s'affiche en plein écran.

2

Toucher un bâtiment sur la carte

Touchez l'emplacement du bâtiment. Le formulaire remonte depuis le bas de l'écran. Le village est détecté automatiquement.

3

Compter les nids

Utilisez les boutons + et − pour chaque type de nid. Maintenez le bouton appuyé pour faire défiler rapidement. Cochez ensuite l'emplacement des nids.

4

Enregistrer et continuer

Appuyez sur Enregistrer l'observation. Le formulaire se ferme et le point apparaît sur la carte. Touchez un nouveau bâtiment pour continuer.

Pour changer de village : exportez d'abord votre CSV, puis appuyez sur l'icône corbeille pour vider les données.
5

Télécharger les données

Appuyez sur le bouton Télécharger dans la barre du haut. Le fichier est enregistré dans vos téléchargements. Vérifiez qu'il est bien sauvegardé avant de vider les données.

Questions fréquentes

Mes données sont-elles perdues si je ferme le navigateur ?
Non. Les données sont sauvegardées automatiquement dans votre navigateur et envoyées vers la base de données du Parc. Elles restent disponibles si vous rouvrez l'outil sur le même appareil et le même navigateur.
Le village détecté est incorrect, que faire ?
Cliquez directement sur le champ Village et tapez le nom correct. La détection automatique utilise OpenStreetMap et peut occasionnellement se tromper pour les très petits hameaux.
L'adresse n'apparaît pas ou est incomplète
L'adresse est récupérée via OpenStreetMap. Si le numéro de maison ne s'affiche pas, il n'est peut-être pas encodé dans OSM pour ce bâtiment. Complétez manuellement le champ adresse.
Comment transférer les données du téléphone vers l'ordinateur ?
Exportez le CSV depuis l'outil mobile, transmettez le fichier par email, puis importez-le dans l'outil bureau via le bouton Importer. Vous pouvez ainsi fusionner plusieurs sessions de terrain.
Je travaille sur plusieurs villages, comment m'organiser ?
Encodez les observations d'un village, exportez le CSV (il porte le nom du village), utilisez le bouton Vider pour repartir de zéro. À la fin, importez tous vos CSV pour les fusionner.
Je n'ai pas de connexion internet sur le terrain
Les outils fonctionnent hors ligne pour l'encodage. Les données sont sauvegardées localement. Le fond de carte et la détection d'adresse nécessitent internet, mais l'encodage lui-même reste possible. Exportez votre CSV et transmettez-le à votre retour.
