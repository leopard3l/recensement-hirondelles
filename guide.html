❓ Guide des outils numériques

Tout ce qu'il faut savoir pour encoder vos observations

🖥 Démarrage rapide — Outil bureau

1

Ouvrir l'outil

Rendez-vous sur l'outil bureau. La carte de la Gaume s'affiche. Les délimitations des entités apparaissent en vert pointillé.

2

Cliquer sur un bâtiment

Cliquez précisément sur le bâtiment où se trouvent les nids. Le formulaire s'ouvre automatiquement. Le village et l'adresse sont détectés automatiquement via GPS (quelques secondes).

3

Remplir le formulaire

Dans l'ordre :

  • Vérifiez le village (corrigez si besoin)
  • Vérifiez l'adresse et complétez le numéro si nécessaire
  • Sélectionnez le type de bâtiment
  • Encodez le nombre de nids (5 colonnes)
  • Cochez l'emplacement des nids
  • Ajoutez une remarque si nécessaire
💡 Le curseur se place automatiquement sur "Nids entiers" à l'ouverture du formulaire. Vous pouvez encoder directement au clavier.
4

Enregistrer

Cliquez sur ✅ Ajouter l'observation. Un point apparaît sur la carte. Les données sont sauvegardées localement ET envoyées automatiquement vers la base de données du Parc.

5

Exporter quand vous avez fini

Cliquez sur ⬇ Télécharger dans la barre du haut. Un fichier CSV est téléchargé au nom recensement_hirondelle_VillageName_2026.csv. Transmettez-le à la personne de contact.

📱 Démarrage rapide — Outil terrain

1

Ouvrir sur votre téléphone

Ouvrez l'outil terrain dans votre navigateur mobile. La carte de la Gaume s'affiche en plein écran.

2

Toucher la carte

Touchez l'emplacement du bâtiment sur la carte. Le formulaire remonte depuis le bas de l'écran. Le village est détecté automatiquement.

3

Compter les nids avec + et −

Utilisez les boutons + et − pour chaque type de nid. Maintenez appuyé pour faire défiler rapidement. Cochez l'emplacement (Sous corniche / Fenêtre).

4

Enregistrer & continuer

Appuyez sur ✅ Enregistrer l'observation. Le formulaire se ferme, le point apparaît sur la carte. Touchez un nouveau bâtiment pour continuer.

💡 Pour changer de village : exportez d'abord votre CSV, puis appuyez sur 🗑 dans la barre du haut pour vider les données.
5

Télécharger vos données

Appuyez sur 💾 Télécharger (X obs.) — le fichier CSV est téléchargé dans vos téléchargements. Vérifiez bien qu'il est enregistré avant de vider les données.

💬 Questions fréquentes

Mes données sont-elles perdues si je ferme le navigateur ?
Non. Les données sont sauvegardées automatiquement dans votre navigateur (localStorage) et envoyées vers la base de données du Parc. Elles restent disponibles quand vous rouvrez l'outil sur le même appareil et le même navigateur.
Le village détecté est incorrect, que faire ?
Cliquez directement sur le champ "Village" et tapez le nom correct. La détection automatique utilise OpenStreetMap et peut parfois se tromper pour les hameaux très petits.
L'adresse n'apparaît pas ou est incomplète
L'adresse est récupérée via OpenStreetMap. Si le numéro de maison ne s'affiche pas, il n'est peut-être pas encore encodé dans OSM pour ce bâtiment. Complétez manuellement le champ adresse.
Comment transférer les données du téléphone vers l'ordinateur ?
Exportez le CSV depuis l'outil mobile (bouton 💾 Télécharger), transmettez le fichier par email ou WhatsApp, puis importez-le dans l'outil bureau (bouton ⬆ Importer) si vous souhaitez tout visualiser ensemble.
Je travaille sur plusieurs villages, comment m'organiser ?
Encodez toutes les observations d'un village, exportez le CSV (il portera le nom du village), puis utilisez le bouton 🗑 Vider pour recommencer avec un nouveau village. À la fin, importez tous vos CSV les uns après les autres pour les fusionner.
Je n'ai pas de connexion internet sur le terrain
Les outils fonctionnent hors ligne pour l'encodage. Les données sont sauvegardées localement. Seul le fond de carte OSM et la détection d'adresse nécessitent internet. L'envoi vers la base de données se fera automatiquement si internet n'était pas disponible — dans ce cas exportez votre CSV et transmettez-le.
