Comment ça fonctionne

Déposez un ou plusieurs fichiers. L'outil détecte automatiquement les colonnes même si elles sont dans un ordre différent ou nommées différemment. Un aperçu est affiché avant l'envoi pour vérifier que tout est correct.

Colonnes reconnues automatiquement : Date (ou Jour), Entité (ou Village, Commune), Observateur, Adresse, Type de bâtiment, les 5 compteurs de nids, Emplacement, Remarque, Latitude, Longitude.

Valeurs NA, N/A, vides → converties en 0 pour les compteurs, en vide pour le texte.
